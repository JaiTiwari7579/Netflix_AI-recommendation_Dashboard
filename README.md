🎬 Netflix AI-Powered Analytics Dashboard

This project is an interactive Netflix analytics dashboard built using Power BI, combined with an AI-based recommendation system developed in Python.

 🚀 Project Overview

- Analyzed 8700+ Netflix titles
- Cleaned and transformed raw Netflix data using Python
- Built a Netflix-themed Power BI dashboard with KPIs and insights
- Implemented an AI-based recommendation system using TF-IDF and cosine similarity

 📊 Key Features

- Total Content, Movies, TV Shows, and Avg Runtime KPIs
- Content distribution by genre, country, year, and rating
- Trend analysis of Netflix content over time
- AI-powered title recommendations based on selected content

 🛠 Tools & Technologies

- Power BI
- Python
- Pandas
- Scikit-learn
- CSV Dataset

🤖 AI Recommendation System

The recommendation system was built using content-based filtering:
- Combined text features such as genre, type, and description
- Applied TF-IDF vectorization
- Calculated cosine similarity between titles
- Generated similar title recommendations for dashboard integration

📂 Project Structure

- `dashboard/` → Power BI dashboard file
- `data/` → cleaned dataset
- `notebooks/` → Python notebook
- `images/` → dashboard screenshots
 💡 Insights

- Movies dominate Netflix content compared to TV Shows
- Content growth increased sharply after 2015
- Drama and international content are among the most common categories
- Runtime and rating patterns help understand audience targeting
